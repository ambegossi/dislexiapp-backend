## :information_source: How To Use

```bash
# Clone this repository
$ git clone https://github.com/ambegossi/dislexiapp-backend

# Go into the repository
$ cd dislexiapp-backend

# Install dependencies
$ yarn

# Start the server
$ yarn dev
```
